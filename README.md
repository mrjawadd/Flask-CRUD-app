# Flask CRUD Web App

A lightweight Flask application that lets you **create, read, update, and delete** items in a database. Perfect for learning Flask basics or quickly building a small web project.

## Key Features

* **Add Items:** Quickly add new entries via a simple form.
* **View Items:** See all your entries in a clean, organized table.
* **Edit Items:** Update any item’s details easily.
* **Delete Items:** Remove items you no longer need.
* **SQLite Backend:** Simple, file-based database—no setup required.

## Getting Started

1. Clone this repo:

   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install the dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:

   ```bash
   python app.py
   ```
4. Open your browser at:

   ```
   http://127.0.0.1:5000
   ```

## How to Use

* Go to the home page to see all items.
* Click **Add** to create a new item.
* Use **Edit** to modify an existing item.
* Click **Delete** to remove items you no longer want.

## Tech Stack
